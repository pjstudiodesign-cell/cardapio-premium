# 🍔 Cardápio Digital Interativo - PJ Studio Design

Este é um sistema de **Cardápio Digital Premium** desenvolvido para proporcionar uma experiência de compra rápida e moderna. Focado em estabelecimentos que buscam profissionalismo e agilidade no atendimento via WhatsApp.

## 🛠️ Funcionalidades
* **Navegação por Abas:** Categorias separadas (Lanches, Bebidas e Doces) facilitando a visualização.
* **Carrinho Inteligente:** Cálculo automático de valores em tempo real.
* **Pedidos via WhatsApp:** Envio de mensagem formatada com todos os dados do cliente e itens escolhidos.
* **Design High-End:** Interface inspirada em padrões SaaS e Apple, com foco em usabilidade móvel.
* **Totalmente Responsivo:** Adaptado para qualquer tamanho de tela de smartphone.

## 🚀 Tecnologias Utilizadas
* **HTML5** (Estruturação)
* **CSS3** (Estilização Premium e Animações)
* **JavaScript** (Lógica de carrinho e integração com API do WhatsApp)
